# User Management React Application

A simple **User Management Web Application** built using **React.js**.  
This project displays user information in a table and allows users to delete individual users or remove all users from the list.

---

## 🚀 Features

- Display user details in a structured table
- Delete individual users
- Delete all users at once
- Clean and responsive UI
- Component-based architecture using React
